# arinc429Analyzer
ARiNC 429 Saleae plugin

note: this project is based on a framework provided by Saleae. [https://github.com/saleae/SampleAnalyzer]


### Linux build

```bash
mkdir build
cd build
cmake ..
cmake --build .
# built analyzer will be located at build/Analyzers/libarinc429Analyzer.so
```
